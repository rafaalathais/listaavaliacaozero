# Lista - avaliação zero

# Requisitos: 
- GCC (compilador C)

# Como rodar:
Pelo terminal navegue até a pasta do projeto (Lista_avaliação_zero) e execute, por exemplo: 

```bash
  gcc fatorial.c -o fatorial
```

Depois, execute:

(Windows)

```bash
  fatorial.exe
```

(Linux / Mac)

```bash
  ./fatorial
```
